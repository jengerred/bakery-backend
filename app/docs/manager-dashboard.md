# 📊 Manager Dashboard — Overview

The Manager Dashboard provides analytics and reporting for bakery managers.

---

## ⭐ Current Features (Phase 1)

- Today’s sales summary
- Payment breakdown
- Recent orders list
- Modular UI components
- `useTodayOrders` hook

---

## ⭐ Planned Features (Phase 2+)

- Sales charts (daily, weekly, monthly)
- Product performance analytics
- Inventory alerts
- Employee performance
- Scheduling
- Payroll
- Exportable reports

---

## ⭐ Architecture Notes

- Built with Next.js (server components)
- Uses shared order history for now
- Will connect to backend analytics API later
