# 🍞 POS System — Overview

The POS is the first completed module of the Bakery Business Suite.  
It is designed to simulate a real cashier workflow with Stripe‑ready payment flows.

---

## ⭐ Features

- Product grid
- Cart system
- Checkout modal
- Cash payments
- Card payments (Stripe)
- Reader simulation
- Receipt printing
- Order history
- Payment method tagging

---

## ⭐ Architecture

- Next.js 14
- React Server Components + Client Components
- TypeScript (strict)
- TailwindCSS
- Stripe Elements
- OrderHistoryContext

---

## ⭐ UI Flow

1. Select products  
2. Add to cart  
3. Open checkout modal  
4. Choose payment method  
5. Complete payment  
6. Generate receipt  
7. Save order to history
