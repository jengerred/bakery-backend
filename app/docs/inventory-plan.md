# 📦 Inventory System — Plan

The inventory system manages ingredients, stock levels, and usage.

---

## ⭐ Phase 2 Goals

- Ingredient list
- Ingredient deduction per order
- Real‑time inventory updates
- Low‑stock alerts
- Auto‑reorder logic
- Ingredient usage reports

---

## ⭐ Data Model (Draft)

- Ingredients
- ProductIngredients (recipes)
- Inventory
- IngredientUsage

---

## ⭐ Workflow

1. Order placed  
2. System calculates ingredient usage  
3. Deduct from inventory  
4. Log usage  
5. Trigger alerts if needed
